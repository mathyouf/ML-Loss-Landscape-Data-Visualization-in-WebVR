The problem

The premise of every machine learning problem is:

  • Someone gives us some data that was generated according to a secret formula. This data could be a bunch of points (that are generated based on some math equation), but could also be fun, like images (the secret formula could be "all of these dogs are golden retrievers") or sounds ("all of these sounds are made by a bird")
  • By looking at this data we were given, we have to figure out how it was generated, so that we can correctly predict a future data point. For example, if we're given a photo, we can say whether it's a golden retriever, or if we are given a sound, we can tell if it was made by a bird.

This specific example follows that pattern: we are given a bunch of data points according to a cubic polynomial ( i.e. generated from y = a * x ^ 3 + b * x^2 + c * x + d), but hasn't told us the values of these a,b,c,d coefficients. Our goal is to learn these coefficients, so that if we're given a new x, we can say what the y should be.

The solution

Most machine learning algorithms follow this pattern.

  1. We have to figure out the "features" of the secret formula that generated the data we were given, so that we can learn them. For our example, those features are the coefficients in the polynomial. For something more complex like the "is this dog a golden retriever" problem, we'd have to look at pixels and colours and formations.
  2. Once we figure out these features (in our case, those a,b,c,d coefficients), we initialize them to some random values. We could now use them to make predictions, but they would be teeeeeerrible because they're just random.
  3. (I'm just going to use our actual example from now on and not dogs)
  4. We start looking at every piece of (x,y) data point we were given. We take the x and based on these default coefficients we have created, we predict what the y would be. WE then look at the correct y from the data point, calculate our error, and then adjust our coefficients so that our value would be closer to the actual one. This part is the most important one and TensorFlow does this for us
  5. We keep doing this for every data point we have, and repeat the entire process a bunch of iterations, and at the end we've pretty much learnt the coefficients!

The fun demo

The blue dots are the training points we were given. The red dots would be our guesses, based on our initial, default coefficients (hella incorrect!). Once you click the train button, the green dots show how our coefficients are getting better. Check what happens if we are given fewer data points or fewer iterations!
